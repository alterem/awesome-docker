# ⚙️ 开发工具类

此目录用于存放各种开发相关的服务和工具。

### Gitea

Gitea 是一个轻量级的代码托管解决方案，类似于 GitHub，但可以自托管。它易于安装和维护，资源占用少，适合个人或小型团队使用。

*   **推荐镜像:** `gitea/gitea:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/r/gitea/gitea](https://hub.docker.com/r/gitea/gitea)
*   **GitHub 链接:** [https://github.com/go-gitea/gitea](https://github.com/go-gitea/gitea)

### MySQL

MySQL 是一个流行的开源关系型数据库管理系统，广泛应用于各种 Web 应用程序。

*   **推荐镜像:** `mysql:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/_/mysql](https://hub.docker.com/_/mysql)
*   **GitHub 链接:** [https://github.com/mysql/mysql-server](https://github.com/mysql/mysql-server)

### Redis

Redis 是一个开源的内存数据结构存储，可用作数据库、缓存和消息代理。

*   **推荐镜像:** `redis:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/_/redis](https://hub.docker.com/_/redis)
*   **GitHub 链接:** [https://github.com/redis/redis](https://github.com/redis/redis)

### PostgreSQL

PostgreSQL 是一个功能强大的开源对象关系型数据库系统，以其可靠性、功能健壮性和性能而闻名。

*   **推荐镜像:** `postgres:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/_/postgres](https://hub.docker.com/_/postgres)
*   **GitHub 链接:** [https://github.com/postgres/postgres](https://github.com/postgres/postgres)

### MongoDB

MongoDB 是一个流行的开源文档数据库，属于 NoSQL 数据库，以其灵活性和可伸缩性而受到青睐。

*   **推荐镜像:** `mongo:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/_/mongo](https://hub.docker.com/_/mongo)
*   **GitHub 链接:** [https://github.com/mongodb/mongo](https://github.com/mongodb/mongo)

### RabbitMQ

RabbitMQ 是一个开源的消息代理，实现了高级消息队列协议（AMQP），用于在应用程序之间进行异步通信。

*   **推荐镜像:** `rabbitmq:management`
*   **Docker Hub 链接:** [https://hub.docker.com/_/rabbitmq](https://hub.docker.com/_/rabbitmq)
*   **GitHub 链接:** [https://github.com/rabbitmq/rabbitmq-server](https://github.com/rabbitmq/rabbitmq-server)

### Kafka

Apache Kafka 是一个分布式流处理平台，用于构建实时数据管道和流式应用程序。

*   **推荐镜像:** `bitnami/kafka:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/r/bitnami/kafka](https://hub.docker.com/r/bitnami/kafka)
*   **GitHub 链接:** [https://github.com/apache/kafka](https://github.com/apache/kafka)

### Jenkins

Jenkins 是一个开源的自动化服务器，用于实现持续集成（CI）和持续交付（CD）流程。

*   **推荐镜像:** `jenkins/jenkins:lts`
*   **Docker Hub 链接:** [https://hub.docker.com/r/jenkins/jenkins](https://hub.docker.com/r/jenkins/jenkins)
*   **GitHub 链接:** [https://github.com/jenkinsci/jenkins](https://github.com/jenkinsci/jenkins)

### PortainerAgent

Portainer Agent 是 Portainer 的一个轻量级代理，用于管理远程 Docker 环境，如 Swarm 集群或独立的 Docker 主机。

*   **推荐镜像:** `portainer/agent:latest`
*   **Docker Hub 链接:** [https://hub.docker.com/r/portainer/agent](https://hub.docker.com/r/portainer/agent)
*   **GitHub 链接:** [https://github.com/portainer/agent](https://github.com/portainer/agent)

### NexusRepositoryManager

Nexus Repository Manager 是一个通用的仓库管理器，支持多种包格式，如 Maven、npm、Docker 等，用于管理和分发二进制文件。

*   **推荐镜像:** `sonatype/nexus3`
*   **Docker Hub 链接:** [https://hub.docker.com/r/sonatype/nexus3](https://hub.docker.com/r/sonatype/nexus3)
*   **GitHub 链接:** [https://github.com/sonatype/nexus-public](https://github.com/sonatype/nexus-public)
